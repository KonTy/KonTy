## 👾 About Me

> I build operating systems, not just apps.

I'm the creator of **smplOS** — a minimal Arch Linux distro built around one idea: *simplicity*. Under 800 MB RAM on cold boot. Fully offline installs in under 2 minutes. One theme command to recolor the entire desktop. A full Rust GUI app suite, a patched suckless terminal, and a forked file manager that upstream refused.

It started as a lighter Omarchy. It became a completely different OS.
